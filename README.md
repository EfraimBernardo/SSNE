Sistema de Seleção de Novos Estudantes.

Criado com HTML, CSS e JavaScript para o Front-End e NodeJS com PostgreSQL para o Back-End.

Este Sistema, permite o controle de seleção de novos estudantes para a instituição, de modo a que a sua nota é analisada e é calculada uma média que, se menor ou igual a 14, é realizado um exame de admissão, caso for menor ou ihual a 20, o aluno é aprovado de forma directa, caso menor que 10, não será apto.
